# Books to Scrape - Learning Project

## Project Overview

This project was created as part of my learning journey in Python and Web Scraping.

The goal is to extract book information from a website, clean the data, and prepare it for analysis and visualization.

## Technologies Used

* Python
* Requests
* BeautifulSoup
* Pandas
* Power BI

## Data Extracted

* Book Title
* Price
* Rating
